# XHTML-Elemente, die in von *nv_epub* erzeugten EPUBs vewendet werden 

| XHTML                   | *Writer* Vorlagenname                           | Beschreibung                                           |
|-------------------------|-------------------------------------------------|--------------------------------------------------------|
| **h1**                  | **Überschrift 1**                               | Teileüberschrift.                                      |
| **h2**                  | **Überschrift 2**                               | Kapitelüberschrift.                                    |
| **h4**                  | **Überschrift 4**                               | Abschnittstrenner.                                     |
| **p**                   | **Standard**                                    | (Nicht explizit angeandt)                              |
| **p.title**             | **Titel**                                       | Romantitel.                                            |
| **p.author**            | **Untertitel**                                  | Autor.                                                 |
| **p.epigraph**          | **Motto**                                       | Motto.                                                 |
| **p.epigraph_source**   | **Motto-Quelle**                                | Die Quelle des Mottos, falls vorhanden.                |
| **p.chapter_beginning** | **Kapitelanfang**                               | Erster Absatz im Kapitel.                              |
| **p.text_body**         | **Textkörper**                                  | Erster Absatz im Abschnitt, falls nicht Kapitelanfang. |
| **p.first_line_indent** | **Textkörper Einzug**                           | Normaler Absatz.                                       |
| **p.quotations**        | **Zitat                                         | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **p.custom_5**          | **Überschrift 5**                               | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **p.custom_6**          | **Überschrift 6**                               | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **p.custom_7**          | **Überschrift 7**                               | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **p.custom_8**          | **Überschrift 8**                               | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **p.custom_9**          | **Überschrift 9**                               | Absatz, der sich visuell vom Textkörper unterscheidet. |
| **ul**                  | (N/A)                                           | Ungeordnete Liste (schließt Listenelemente ein).       |
| **li**                  | (**Textkörper Einzug** wird angewandt)          | Absatz eines Listenelements.                           |
| **em**                  | **Betont**                                      | Betonter Text.                                         | 
| **strong**              | **Stark betont**                                | Stark betonter Text.                                   |
